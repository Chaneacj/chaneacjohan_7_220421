## Présentation
Créer un réseau social d'entreprise.

## Technologies utilisées
▹ HTML
▹ CSS
▹ JavaScript
▹ Vue.js
▹ Node.js
▹ Express.js
▹ MySQL
▹ SQL

## Prérequis :
```
Vue.js, Node.js et MySQL.
```

## Installation :


### Back-end 
Démarrer le serveur 
```
nodemon server
```

### Base de données
Créer base de données
```
Remplir le config.json
```

### Frontend
Démarrer le serveur 
```
npm run serve
```
Rendez-vous sur `http://localhost:8080/`
